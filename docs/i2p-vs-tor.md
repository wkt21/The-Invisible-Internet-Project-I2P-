I2P vs Tor
•	Purpose: Tor primarily anonymizes access to the public internet and supports exit nodes; I2P is optimized for internal, self-contained services.[i2p +1]
•	Routing: Tor uses onion routing with bidirectional circuits; I2P uses unidirectional tunnels and garlic routing, which changes performance and threat models.[i2p +1]
•	Use cases: Tor for anonymous clearnet browsing, I2P for hosting eepsites, P2P apps, and internal services.
