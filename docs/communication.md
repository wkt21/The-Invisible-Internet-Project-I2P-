How to communicate on I2P
•	Eepsites (.i2p): internal websites reachable by browser proxies configured to 127.0.0.1:4444.[i2p]
•	IRC: Irc2P via local IRC proxy (default localhost:6668).[i2p]
•	Email: SusiMail and other I2P mail services operate entirely inside the network.[i2p]
•	File sharing: I2PSnark (built-in BitTorrent client) and other I2P-aware clients.[i2p]
Operational note: register readable names via address book services (stats.i2p, no.i2p) or use Base64 destinations.
