How to install and access I2P
•	Install: Use the official packages or the Easy Install Bundles for Windows/macOS; Java-based reference implementation is the recommended client.[i2p +1]
•	Router console: After install open http://127.0.0.1:7657 to complete setup and view status.[i2p]
•	Browser configuration: set HTTP proxy to 127.0.0.1 port 4444 (Firefox recommended for per-profile proxy settings). Use a dedicated browser profile for I2P browsing.[i2p]
•	Bootstrapping: allow several minutes for reseeding and hours-to-days of continuous uptime for optimal performance. Keep the router running and allocate adequate shared bandwidth.
